# roadmap-api

![Go Version](https://img.shields.io/badge/Go-1.25-blue)
![Fiber Version](https://img.shields.io/badge/Fiber-v3.0.0--rc.3-green)

Простое API выводящее список роадмапов и их задачи

## Требования

-   go 1.25

## Установка и запуск

1. Установите зависимости
    ```
     go mod download
    ```
2. Запустите сервер
    ```
     go run main.go
    ```
3. Сервер будет доступен по адресу `http://localhost:3000`
